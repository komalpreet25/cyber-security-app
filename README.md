# 🔐 AI Powered Cyber Security Assistant

A beginner-friendly Cyber Security application built using Python and Streamlit. The project helps users improve online safety through password analysis, phishing detection, URL checking, and cyber security awareness tips.

## 🚀 Features

### 🔐 Password Strength Checker

* Checks password strength based on:

  * Length
  * Uppercase letters
  * Numbers
  * Special characters
* Provides a security score and improvement tips.

### 🕵️ Phishing Detector

* Analyzes messages and emails for suspicious phishing keywords.
* Generates a risk score.
* Helps users identify common phishing attempts.

### 🌐 URL Safety Checker

* Performs basic URL safety analysis.
* Checks whether a website uses HTTPS.
* Alerts users about potentially suspicious URLs.

### 🤖 AI Cyber Assistant

* Answers basic cyber security questions.
* Provides guidance on:

  * Password security
  * Phishing awareness
  * Safe browsing practices

## 🛠️ Technologies Used

* Python
* Streamlit
* Regular Expressions (Regex)
